### Environment variables

`VUE_APP_BACK_URL` should be the API origin only, for example:

```bash
VUE_APP_BACK_URL=https://example.com
```

The app trims any trailing `/` before building requests, so both `https://example.com` and `https://example.com/` resolve to the same API base URL.
